# Task 2: Sales Performance Analysis

## Objective
Analyze sales performance across regions, categories, products, and time using the Superstore Sales dataset.

## Dataset
Sample Superstore Sales Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## KPIs Created
- Total Revenue
- Total Profit
- Profit Margin
- Average Order Value
- Total Orders
- Conversion Metrics: Not available in dataset

## Analysis Performed
- Data Cleaning
- Monthly Sales Trend
- Monthly Profit Trend
- Regional Sales Performance
- Regional Profit Performance
- Category-wise Sales and Profit
- Sub-Category Sales
- Top 10 Products by Sales
- Bottom 10 Products by Profit
- Discount vs Profit Analysis
- Seasonality Analysis

## Tactical Recommendations
1. Focus marketing on high-performing products and categories.
2. Reduce excessive discounts on low-profit products.
3. Improve inventory planning before peak sales months.
4. Create region-specific sales campaigns.
5. Use customer loyalty programs to increase repeat purchases.

## Deliverables
- Jupyter Notebook with analysis and charts
- PDF summary report
- Cleaned dataset
