# Task 3 - Website Traffic Analysis

## Objective

Analyze website traffic logs to understand user journeys, landing pages, bounce rates and user engagement.

## Dataset

Website Traffic Analysis Dataset (Kaggle)

## Project Workflow

- Data Cleaning
- Traffic Analysis
- Session & User Metrics
- Bounce Rate Analysis
- User Journey Analysis
- Entry & Exit Page Analysis
- Business Recommendations

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations

- User Journey Funnel
- Top Entry Links
- Estimated Exit Links
- Top Traffic Sources

## Key Insights

- Cleaned website traffic logs
- Calculated sessions and users
- Estimated bounce rate
- Estimated session duration
- Identified top landing pages
- Identified high drop-off pages

## Recommendations

1. Improve high-traffic landing pages.
2. Reduce bounce rate with better content.
3. Optimize CTAs.
4. Focus on high-performing traffic locations.
5. Improve analytics tracking.
